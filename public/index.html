BeeAPI Go

Welcome to the BeeAPI Go server. This is a Go implementation of the BeeAPI service for AlgoHive puzzles.

API Documentation GitHub

Getting Started

To use this API:

  1. Browse the API documentation at /swagger/index.html
  2. Use the /auth/login endpoint to authenticate
  3. Explore themes and puzzles with the provided endpoints

Default Credentials

If no environment variables are set, the default user is:

  • Username: admin
  • Password: admin
